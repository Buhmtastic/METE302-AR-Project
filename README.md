# METE302-AR_Project

## About This Project

This project was created as part of a practical lab for the **XR Convergence Project** course, taught by **Professor Sanghun Park**, Vice Dean of the **Sogang Graduate School of Virtual Convergence**.

## Technical Details

- **Unity Version**: Unity 6 (version 6000.0.34f1)
- **AR Implementation**: Both marker-based AR and markerless AR were implemented and practiced in this project

## Project Overview

AR Project implementing various augmented reality techniques using Unity 6
